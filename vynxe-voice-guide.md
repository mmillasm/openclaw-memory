# 🗣️ VYNXE — GUÍA DE VOZ POR CANAL

> Cómo hablar como VYNXE en cada contexto: emails, Instagram, landing, WhatsApp
> Aplica a todas las verticales: Inmobiliarias, Salud, Retail, Services

---

## 🎯 PRINCIPIOS UNIVERSALES (Aplican a todo)

1. **Directo:** Si puedes decirlo en 5 palabras, no uses 20
2. **Útil:** Cada mensaje debe aportar valor concreto
3. **Humano:** Como una conversación, no un manual corporativo
4. **Sin jerga:** "Clientes" no "leads", "responde" no "automatiza flujos"

---

## 📧 EMAILS (Cold Email + Follow-ups)

### Tono General Emails
- **Profesional pero cercano:** No eres un banco, pero tampoco un amigo de fiesta
- **Respetuoso del tiempo:** Reconoce que están ocupados
- **Investigado:** Muestra que sabes quién son
- **Sin presión:** Exploración, no venta forzada

### Estructura Email Frío (Template General)

```
ASUNTO: Específico, personal, curiosidad

Hola [Nombre],

[CONEXIÓN: Máx 2 líneas mostrando que investigaste]
"Veo que [Agencia/Clínica/Tienda] [actividad específica]."

[PROBLEMA: Máx 2 líneas, empático]
"[Problema específico del vertical]."

[SOLUCIÓN: Máx 3 líneas, clara]
"Ayudamos a [tipo de negocio] como el suyo a tener un asistente virtual que [beneficio principal]."

[CTA: Una opción, fácil decir sí]
"¿Les interesaría una conversación de 20 minutos para ver si esto tiene sentido para ustedes?"

[DESPEDIDA: Simple]
"Saludos,
[Tu nombre]"
```

## 💬 WHATSAPP (Respuestas del Bot — Por Vertical)

### Tono del Asistente Virtual
- **Cálido pero eficiente:** Amable, pero va al grano
- **Natural:** Emojis ocasionales, no exagerados
- **Proactivo:** Sugiere el siguiente paso
- **Humano:** Si no sabe, lo dice y escala

---

## 💬 WHATSAPP — VYNXE INMOBILIARIAS

#### Cliente: "Hola, vi una propiedad"
```
¡Hola! 👋 

Veo que te interesó una propiedad. ¿Me puedes decir cuál? Puedes:
• Pegar el título del aviso
• Decirme la dirección aproximada
• O describirla (ej: "depto 2D 2B en Las Condes")
```

#### Cliente: "¿Cuánto cuesta?"
```
Depende de la propiedad que te interese.

¿Tienes una específica en mente? Así te puedo dar información exacta.

Si no, dime:
• ¿Compras o arriendas?
• ¿En qué zona buscas?
• ¿Cuántos dormitorios necesitas?

Y te muestro opciones que se ajusten a tu presupuesto 🏠
```

#### Cliente: "Quiero agendar una visita"
```
¡Perfecto! 📅

Para agendar, necesito saber:

1. ¿Qué propiedad quieres visitar? (puedes describirla)
2. ¿Qué día te funciona mejor?
3. ¿Mañana o tarde?

Te confirmo en segundos con la disponibilidad actual.
```

## 📄 PDF/PROPUESTAS (Documentos formales)

### Tono Propuestas
- **Profesional pero accesible:** Más formal que Instagram, menos que un contrato legal
- **Estructurado:** Secciones claras, fácil de escanear
- **Orientado a resultados:** Cada sección conecta con el beneficio
- **Transparente:** Precios claros, sin letra chica

### Estructura Propuesta Comercial

```
1. PORTADA
   Logo vynxe [vertical]
   "Propuesta para [Nombre Negocio]"
   Fecha

2. EL PROBLEMA (1 página)
   "Entendemos que..."
   Datos específicos de su situación

3. LA SOLUCIÓN (2 páginas)
   Cómo funciona vynxe [vertical]
   Beneficios específicos para ellos

4. PROPUESTA DE VALOR (1 página)

   Qué incluye

   CTA: "¿Empezamos?"

   Tu nombre, email, teléfono
```

### Antes de enviar un EMAIL:
- [ ] ¿El asunto es específico y personal?

- [ ] ¿El hook captura atención en 2 segundos?
- [ ] ¿La story tiene problema → solución → resultado?
- [ ] ¿Usas emojis pero no exageras?
- [ ] ¿El CTA es específico (no "link en bio" genérico)?
- [ ] ¿Los hashtags son relevantes y específicos del vertical?

- [ ] ¿El headline provoca o promete?

- [ ] ¿Hay suficientes CTAs (no solo al final)?
- [ ] ¿Todo el texto es escaneable (bullet points, no párrafos largos)?

- [ ] ¿Las respuestas son naturales (no robóticas)?
